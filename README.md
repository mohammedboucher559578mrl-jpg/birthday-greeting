# 🎂 生日快乐祝福网站

基于 [arunboy/love](https://github.com/arunboy/love) 表白网站框架改编，献给特别的生日祝福。

## 网站预览

👉 https://mohammedboucher559578mrl-jpg.github.io/birthday-greeting/

## 框架来源

- 原框架：[arunboy/love](https://github.com/arunboy/love)（MIT License）
- 采用 `flowtime.js` 流式翻页框架
- 心形粒子特效来自原项目

## 文件结构

```
/
├── index.htm       # 主页面
├── all.min.css     # 样式
├── all.min.js      # flowtime 框架
├── jquery.min.js   # jQuery 依赖
├── love.min.js     # 心形粒子特效
├── lovebgm.mp3     # 背景音乐
└── img/            # 图片素材
```

## 使用说明

按键盘 `↓` 或滑动屏幕翻页浏览祝福内容。
